# Ale&Scale
**Ale&Scale** - це своєрідний посібник у світі слабоалкогольних продуктів. Він створений для тих, хто хоче дізнатися історію та особливості виробництва напоїв, а також знайти свій ідеальних смак без зайвих зусиль.

## Викорситані технології:
- HTML5
- CSS3
- Flexbox / Grid
- Adaptive design
- JavaScript (ES6+)
- Images optimization
  
### Робота з DOM (Document Object Model)
- Динамічне створення контенту: Скрипт автоматично генерує та додає поточну дату у футер сайту (`createElement`, `appendChild`).
- Маніпуляція елементами: Реалізовано додавання вітального блоку на головну сторінку через `append()`.
- Зміна стилів: JS динамічно змінює CSS-класи елементів для реалізації візуальних ефектів (наприклад, підсвітка меню та активний стан хедера).

### Обробка подій
- Click Events:
  - Кнопка Змінити тему (Dark/Light Mode).
  - Акордеон Читати місію на головній сторінці (розгортання/згортання тексту).
- Mouse Events: Реалізовано кастомну підсвітку пунктів меню при наведенні (`mouseenter`, `mouseleave`).
- Keyboard Events: Реалізовано доступність — зміна розміру шрифту на сторінці клавішами `ArrowUp` та `ArrowDown` (`keydown`).
- Form Events: Перехоплення події відправки форми (`submit`) для проведення валідації даних.

### Робота з LocalStorage
- Збереження теми: Обрана користувачем тема (світла/темна) зберігається в пам'яті браузера і автоматично застосовується при наступному візиті.
- Дані користувача: При успішній реєстрації ім'я користувача зберігається в `localStorage` для подальшої ідентифікації та персоналізації інтерфейсу (наприклад, вітання у профілі).

### Валідація форм
- Login: Перевірка формату email та довжини пароля.
- Registration: Перевірка співпадіння полів "Пароль" та "Підтвердіть пароль".
- Блокування відправки форми та виведення попереджень (`alert`), якщо дані введено некоректно.

## Як запустити:
1. Скачайте або клонуйте репозиторій.  
2. Відкрийте `index.html` у будь-якому браузері.

## Структура:
```
/ale-and-scale
├─ /css
│   ├─ /about-beers.css    # Стилі статей
│   ├─ /base.css           # Базові стилі, змінні, шрифти
│   ├─ /catalog.css        # Стилі сторінки каталогу
│   ├─ /components.css     # Компоненти інтерфейсу
│   ├─ /index.css          # Стилі головної сторінки
│   ├─ /layout.css         # Макет сторінки, контейнери, сітка, навігація
│   └─ /styles.css         # Головний файл з @import
│
├─ /js
│   ├─ /pages
│   │    ├─ /index.js
│   │    ├─  /profile.js
│   │    └─ /registration.js
│   └─ /main.js    
├─ /images                 # Папка із зображеннями
│   ├─ /articles           # Зображення для статей
│   └─ /beer-style         # Зображення для каталогу пива
│
├─ about-beer.html         # Стаття про пиво
├─ about-craft-beer.html   # Стаття про крафтове пиво
├─ catalog.html            # Каталог стилів пива
├─ index.html              # Головна сторінка
├─ profile.html              # Сторінка входу
└─ registration.html       # Сторінка реєстрації
```
